类别|名称|时间复杂度|空间复杂度|是否稳定|
:-|:-|:-|:-|:-
归并|**归并排序**|O(nlogn)|O(n)|稳定|
交换|**快速排序**|O(nlogn)|O(logn)~O(n)|不稳定|
交换|冒泡排序|O(n^2)|O(1)|稳定|
插入|希尔排序|O(n^1.3)|O(1)|不稳定|
插入|直接插入排序|O(n^2)|O(1)|稳定|
选择|直接选择|O(n^2)|O(1)|不稳定|
选择|堆排序|O(nlogn)|O(1)|不稳定|
